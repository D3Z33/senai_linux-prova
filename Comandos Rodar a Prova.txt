Desativar o Firewall
sudo ufw disable

Ativar o firewall com permissões de saída
sudo ufw default allow outgoing

Desativar o firewall com permissões de entrada
sudo ufw default deny incoming 

Ativar as regras
sudo ufw enable 


Para a prova poder rodar
chmod +x ./prova_linux.sh


Para executar a prova
sudo bash ./prova_linux.sh
